# BMI_Calculator
A BMI Calculator computes Body Mass Index using height and weight to determine health categories like underweight, normal, overweight, or obese. It’s a simple and efficient tool for fitness tracking, offering instant results and easy implementation for beginners learning programming logic.
A simple application to calculate Body Mass Index (BMI) using height and weight. It helps users determine their health category, such as underweight, normal, overweight, or obese, based on BMI values.

## Features
- Easy input for height and weight
- Calculates BMI instantly
- Provides health classification
- Supports metric units (kg, meters)

## How to Use
1. Clone this repository.
2. Run the application.
3. Enter your height and weight.
4. Get your BMI result and health category.

## Technologies
- Tkinter(Python) or Any tech stack according to the requirement
